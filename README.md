# Device Model Indicator Library

本仓库用于存放姿轨控单机设备模型指标库 Excel 文件，以及对应的使用说明文档。

## 文件目录

指标库文件统一放在 `device-model-indicators/` 目录下。

| 文件 | 单机类别 | 说明 |
| ---- | -------- | ---- |
| `星敏_指标库_20260225.xlsx` | 星敏感器 | 星敏型号、测量精度、视场、保护角、功耗、寿命等指标 |
| `陀螺_指标库_20260225.xlsx` | 陀螺仪 | ARW、零偏稳定性、常值零偏、标度因数、量程、带宽等指标 |
| `磁强计_指标库_20260225.xlsx` | 三轴磁强计 | 量程、噪声、零偏、软磁/硬磁误差、非正交误差等指标 |
| `太敏_指标库_20260225.xlsx` | 太阳敏感器 | 视场、安装倾角、通道误差、通道噪声、漂移等指标 |
| `飞轮_指标库_20260225.xlsx` | 反作用飞轮 | 最大角动量、最大力矩、最高转速、功耗、质量、寿命等指标 |
| `磁力矩器_指标库_20260226.xlsx` | 磁力矩器 | 额定磁矩、剩磁、响应时间、功耗、质量、寿命等指标 |

## 使用方式

1. 在 GitHub 页面进入 `device-model-indicators/` 目录。
2. 点击需要的 `.xlsx` 文件。
3. 点击 `Download raw file` 下载到本地。
4. 在 MATLAB 单机模型加载脚本中选择对应 Excel 文件和型号名称。

陀螺加载示例：

```matlab
cd 单机模型和指标库
[gyroPm, GyroDB] = DM_Gyr_Load('LX630GYR', '陀螺_指标库_20260225.xlsx');
```

注：陀螺指标库中的型号可带技术类型后缀，例如 `LX630GYR(FOG)`；调用 `DM_Gyr_Load` 时使用剥离后缀后的 key，例如 `LX630GYR`。

## 命名约定

1. 项目等效型号以前缀开头，例如 `LX630`、`YYMQ9`。
2. 星敏型号显式包含 `STT`。
3. 陀螺型号保留技术类型后缀，例如 `(FOG)`、`(MEMS)`。
4. 飞轮、磁力矩器、太敏、磁强计等型号可体现关键能力值。
5. 型号名中的小数使用 `p` 表示，例如 `2p4Nms` 表示 `2.4Nms`。

## 更新流程

1. 在本地修改对应 `DM_*_Expt.m` 源表。
2. 使用 MATLAB 重新生成 Excel 指标库和 Markdown 汇总。
3. 运行对应 `DM_*_Load.m` 校验新增型号可读取。
4. 将更新后的 `.xlsx` 文件和说明文档提交到本仓库。

## 注意事项

1. 本仓库中的项目等效型号用于仿真、型号选型和管理软件适配，不等同于厂家实物型号。
2. 后续新增型号时，应保持 Excel 文件名和目录结构稳定，避免影响加载脚本和管理软件引用。
