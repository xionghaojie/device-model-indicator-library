# 姿轨控单机模型指标库使用说明

## 1. 目录结构

本仓库的指标库 Excel 文件放置在 `device-model-indicators/` 目录中，说明文档放置在 `docs/` 目录中。

## 2. 指标库文件

| 文件 | 单机类别 |
| ---- | -------- |
| `星敏_指标库_20260225.xlsx` | 星敏感器 |
| `陀螺_指标库_20260225.xlsx` | 陀螺仪 |
| `磁强计_指标库_20260225.xlsx` | 三轴磁强计 |
| `太敏_指标库_20260225.xlsx` | 太阳敏感器 |
| `飞轮_指标库_20260225.xlsx` | 反作用飞轮 |
| `磁力矩器_指标库_20260226.xlsx` | 磁力矩器 |

## 3. MATLAB 加载方式

将 Excel 文件放在 MATLAB 当前工作目录，或放在 `DM_*_Load.m` 可访问的路径下，然后按型号名称读取。

```matlab
[stkPm, StkDB] = DM_Stk_Load('LX630STT', '星敏_指标库_20260225.xlsx');
[gyrPm, GyrDB] = DM_Gyr_Load('LX630GYR', '陀螺_指标库_20260225.xlsx');
[whePm, WheDB] = DM_Whe_Load('LX630WHL2p4Nms', '飞轮_指标库_20260225.xlsx');
```

陀螺 Excel 表内的型号名称可以包含技术类型后缀，例如 `YYMQ9GYR(FOG)`；加载函数使用的 key 为 `YYMQ9GYR`。

## 4. 维护流程

1. 修改本地单机模型源表 `DM_*_Expt.m`。
2. 执行对应导出脚本生成 Excel 指标库。
3. 使用 `DM_*_Load.m` 做型号读取检查。
4. 将 Excel 指标库和说明文档同步到 GitHub。

## 5. 适用边界

本仓库中的型号用于姿轨控仿真、指标对照、管理软件适配和工程选型沟通。若用于正式任务设计，应结合任务书、接口数据单、热控供电约束和姿控总体设计再次确认。
